# PARTE C — Questões de síntese

# 1. A apresentação afirma que "instruções de I/O são privilegiadas".
#    Como um programa de usuário consegue fazer I/O sem executar
#    instruções privilegiadas diretamente? (mínimo 2 frases)

Resposta 1:
Um programa de usuário faz I/O usando chamadas de sistema, como write(), que solicitam serviços ao kernel. O kernel executa as instruções privilegiadas em modo kernel e controla o acesso seguro aos dispositivos de hardware.


# 2. Compare write(fd, buffer, tamanho) com o "programa de canal"
#    da apresentação: o que os dois têm em comum em termos de
#    responsabilidades?

Resposta 2:
Os dois descrevem ao sistema o que deve ser transferido, para onde e qual o tamanho dos dados. Tanto o write() quanto o programa de canal delegam ao sistema operacional ou controlador a execução da operação de entrada e saída.
